# Accessibility Widget — Drupal 10/11 Module

<a id="english"></a>

> Drupal 10/11 module that loads the BAUER GROUP Accessibility Widget.

**🇬🇧 English** · [🇩🇪 Deutsch](#-deutsch)

```bash
composer require bauer-group/accessibility-widget-drupal
drush en accessibility_widget
```

Copy the widget assets from `packages/widget/dist/` to `modules/contrib/accessibility_widget/assets/`.

Configuration: `/admin/config/user-interface/accessibility-widget` (assumes a form stub — to be delivered as a ConfigForm in v1.1).
