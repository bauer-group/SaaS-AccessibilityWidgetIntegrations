# Accessibility Widget — WordPress Plugin

<a id="english"></a>

> WordPress plugin that loads the BAUER GROUP Accessibility Widget (BFSG / EN 301 549 / WCAG 2.2 AA).

**🇬🇧 English** · [🇩🇪 Deutsch](#-deutsch)

## Installation

**No assets to host** — the widget loads from the BAUER GROUP CDN (floating `v1` tag) and stays current automatically.

1. Download `wordpress-accessibility-widget-<version>.zip` from the [GitHub Releases](https://github.com/bauer-group/SaaS-AccessibilityWidgetIntegrations/releases/latest).
2. WordPress admin → **Plugins → Add New → Upload Plugin** → choose the ZIP → **Install** → **Activate**.

That's it — the widget appears on the front end immediately.

## Configuration

**Settings → Accessibility Widget.** Every option is exposed, grouped into **Appearance / Behavior / Content / Features / Advanced** and shown **bilingual (EN + DE)** in your admin language — position, offset, colour, locale, the 15 accessibility features, behaviour, the accessibility-statement URL and advanced self-hosting. Full option list: **[configuration reference](https://github.com/bauer-group/SaaS-AccessibilityWidgetIntegrations/blob/main/docs/configuration.md)**.

## 🇩🇪 Deutsch

> WordPress-Plugin, das das BAUER GROUP Accessibility Widget lädt (BFSG / EN 301 549 / WCAG 2.2 AA).

[🇬🇧 English](#english) · **🇩🇪 Deutsch**

### Installation

**Keine Assets zu hosten** — das Widget lädt vom BAUER GROUP CDN (floating `v1`-Tag) und bleibt automatisch aktuell.

1. `wordpress-accessibility-widget-<version>.zip` von den [GitHub Releases](https://github.com/bauer-group/SaaS-AccessibilityWidgetIntegrations/releases/latest) herunterladen.
2. WordPress-Admin → **Plugins → Installieren → Plugin hochladen** → ZIP wählen → **Installieren** → **Aktivieren**.

Fertig — das Widget erscheint sofort im Frontend.

### Konfiguration

**Einstellungen → Accessibility Widget.** Jede Option ist verfügbar, gruppiert in **Darstellung / Verhalten / Inhalt / Funktionen / Erweitert** und **zweisprachig (EN + DE)** in der Admin-Sprache — Position, Abstand, Farbe, Sprache, die 15 Barrierefreiheits-Funktionen, Verhalten, URL der Barrierefreiheitserklärung und erweitertes Self-Hosting. Komplette Optionsliste: **[Konfigurations-Referenz](https://github.com/bauer-group/SaaS-AccessibilityWidgetIntegrations/blob/main/docs/configuration.md)**.
