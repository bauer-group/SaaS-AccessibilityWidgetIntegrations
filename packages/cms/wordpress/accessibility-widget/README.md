# Accessibility Widget — WordPress Plugin

<a id="english"></a>

> WordPress plugin that loads the BAUER GROUP Accessibility Widget.

**🇬🇧 English** · [🇩🇪 Deutsch](#-deutsch)

Plugin folder `accessibility-widget/` → `wp-content/plugins/`.

Copy the widget assets (`accessibility-widget-loader.min.js`, `accessibility-widget-core.min.js`, `accessibility-widget.min.css`) from `packages/widget/dist/` to `wp-content/plugins/accessibility-widget/assets/`.

Configuration: Settings → Accessibility Widget.

License: MIT
