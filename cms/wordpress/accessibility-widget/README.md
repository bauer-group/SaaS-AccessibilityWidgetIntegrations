# BFSG Widget — WordPress Plugin

Plugin-Ordner `accessibility-widget/` → `wp-content/plugins/`.

Widget-Assets (`accessibility-widget-loader.min.js`, `accessibility-widget-core.min.js`, `accessibility-widget.min.css`) aus `packages/widget/dist/` nach `wp-content/plugins/accessibility-widget/assets/` kopieren.

Konfiguration: Einstellungen → BFSG Widget.
