# BFSG Widget — WordPress Plugin

Plugin-Ordner `bfsg-widget/` → `wp-content/plugins/`.

Widget-Assets (`bfsg-widget-loader.min.js`, `bfsg-widget-core.min.js`, `bfsg-widget.min.css`) aus `packages/widget/dist/` nach `wp-content/plugins/bfsg-widget/assets/` kopieren.

Konfiguration: Einstellungen → BFSG Widget.
