# BFSG Widget — TYPO3 13 Extension

```bash
composer require bauer-group/bfsg-widget-typo3
```

1. Extension aktivieren.
2. Widget-Assets aus `packages/widget/dist/` nach `EXT:bfsg_widget/Resources/Public/JavaScript/` und `Css/` kopieren.
3. TypoScript-Template: "BFSG Widget" als statische Vorlage einbinden.
4. Konstanten über Constant-Editor anpassen (Position, Locale, Markenfarbe, SRI).
