# Accessibility Widget — TYPO3 13 Extension

<a id="english"></a>

> TYPO3 13 extension that loads the BAUER GROUP Accessibility Widget.

**🇬🇧 English** · [🇩🇪 Deutsch](#-deutsch)

```bash
composer require bauer-group/accessibility-widget-typo3
```

1. Activate the extension.
2. Copy the widget assets from `packages/widget/dist/` to `EXT:accessibility_widget/Resources/Public/JavaScript/` and `Css/`.
3. TypoScript template: include "Accessibility Widget" as a static template.
4. Adjust the constants via the Constant Editor (position, locale, brand color, SRI).
