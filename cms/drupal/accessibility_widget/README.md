# Accessibility Widget — Drupal 10/11 Module

```bash
composer require bauer-group/accessibility-widget-drupal
drush en accessibility_widget
```

Widget-Assets aus `packages/widget/dist/` nach `modules/contrib/accessibility_widget/assets/` kopieren.

Konfiguration: `/admin/config/user-interface/accessibility-widget` (Formular-Stub vorausgesetzt — in v1.1 als ConfigForm nachgereicht).
