# BFSG Widget — Drupal 10/11 Module

```bash
composer require bauer-group/bfsg-widget-drupal
drush en bfsg_widget
```

Widget-Assets aus `packages/widget/dist/` nach `modules/contrib/bfsg_widget/assets/` kopieren.

Konfiguration: `/admin/config/user-interface/bfsg-widget` (Formular-Stub vorausgesetzt — in v1.1 als ConfigForm nachgereicht).
